# Goal
This project aims to segment customers based on purchasing behavior and predict future purchasing peaks or churn. By analyzing transaction data, we’ll provide actionable insights for retail strategy.

# Getting started

1. Create virtual environment
    ```
    py -m venv .venv
    ```
2. Select the kernel as virtual enviroment in Customer_Segmentation_and_Crunch_prediction.ipynb
3. Run first code block to install packages
